# NeuralOps Dashboard

A full-stack monitoring dashboard for ML models in production, featuring real-time drift detection.

## Features
- Real-time model performance monitoring.
- Data and concept drift detection.
- Customizable dashboards and alerts.
- Integrates with popular ML frameworks.

## Technologies
- **Frontend:** React, TypeScript, Chart.js
- **Backend:** Node.js, Express, Python (for ML integration)
- **Database:** MongoDB

## Installation

```bash
git clone https://github.com/Wasisange/neural-ops-dashboard.git
cd neural-ops-dashboard
# Install frontend dependencies
cd frontend && npm install && cd ..
# Install backend dependencies
cd backend && npm install && cd ..
# Install ML dependencies
pip install -r requirements.txt
```

## Usage

```bash
# Start backend server
cd backend && npm start
# Start frontend development server
cd frontend && npm start
```
